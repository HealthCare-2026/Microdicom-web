Orthanc Architecture & Platforms

Orthanc เป็น DICOM Server แบบ Open-source ที่ออกแบบมาให้มีขนาดเล็ก (Lightweight) และช่วยลดความซับซ้อนของมาตรฐาน DICOM ทำให้สามารถเชื่อมต่อกับระบบภาพทางการแพทย์ และระบบสารสนเทศของโรงพยาบาลได้ง่ายขึ้น

Core Engine

Orthanc Core Server พัฒนาด้วยภาษา C++ เพื่อให้มีประสิทธิภาพสูง และสามารถทำงานเป็น DICOM Server ขนาดเล็ก (lightweight) ได้

ระบบใช้ DCMTK Toolkit สำหรับการ implement มาตรฐาน DICOM เช่น

  • DICOM C-STORE (รับ/ส่งภาพ)
  • DICOM C-FIND (ค้นหาข้อมูล)
  • DICOM C-MOVE / C-GET (retrieve images)

Architecture Concept

Orthanc acts as a bridge between DICOM systems and modern web technologies.

RESTful API Layer

Orthanc มี REST API ที่ช่วยแปลงการทำงานของ DICOM ให้กลายเป็น Web Service ที่ใช้งานง่ายผ่าน HTTP

นักพัฒนาสามารถเข้าถึงข้อมูลภาพในรูปแบบ JSON, PNG, JPEG หรือ DICOM

  • Retrieve DICOM instances
  • Query study / patient metadata
  • Upload images through HTTP

API นี้ช่วยให้สามารถพัฒนาแอปพลิเคชันด้วยภาษา Python, JavaScript, Java, C#, PHP ได้ง่าย

Storage & Database

Orthanc ใช้ SQLite เป็นค่าเริ่มต้นในการเก็บ metadata index และเก็บไฟล์ภาพ DICOM ใน filesystem

ระบบสามารถขยายได้ผ่าน plugin เพื่อใช้ฐานข้อมูลระดับ enterprise เช่น

  • PostgreSQL
  • MySQL / MariaDB
  • Cloud storage backend

โครงสร้างนี้ช่วยให้ Orthanc สามารถ scale ได้ตั้งแต่ small imaging lab จนไปถึง hospital PACS

Plugin Architecture

Orthanc มี Plugin System ที่ช่วยให้สามารถเพิ่มความสามารถใหม่ โดยไม่ต้องแก้ไข core server

Plugin ที่สำคัญใน ecosystem เช่น

  • DICOMweb plugin
  • Web Viewer (Orthanc Explorer)
  • PostgreSQL database plugin
  • AI / research integration plugins

โครงสร้างนี้ทำให้ Orthanc สามารถปรับใช้ได้ใน research, hospital PACS, AI imaging pipelines

Orthanc System Architecture

Orthanc เป็น DICOM Server ที่ถูกพัฒนาด้วยภาษา C++ และออกแบบให้เป็นระบบที่มีขนาดเล็ก (lightweight) แต่สามารถขยายความสามารถได้ผ่าน plugin ecosystem ทำให้สามารถนำไปใช้งานได้ทั้งในระบบ PACS ของโรงพยาบาล งานวิจัยทางการแพทย์ และระบบ Medical Imaging

  • Orthanc Core Server ทำหน้าที่เป็นศูนย์กลางในการจัดการ workflow ของข้อมูลภาพทางการแพทย์ และการสื่อสารระหว่างระบบ DICOM
  • DCMTK Toolkit ใช้สำหรับ implement DICOM protocol เช่น C-Store, C-Find และ C-Move
  • Mongoose HTTP Server ให้บริการ REST API เพื่อให้ Web applications สามารถเข้าถึงข้อมูลภาพได้ผ่าน HTTP
  • SQLite ใช้เก็บ metadata index ของภาพ ในขณะที่ไฟล์ DICOM จะถูกจัดเก็บไว้ใน filesystem
  • Orthanc Explorer เป็น Web interface สำหรับดูภาพทางการแพทย์ผ่าน browser โดยไม่ต้องติดตั้ง software เพิ่ม

Supported Platforms

Orthanc สามารถติดตั้งและใช้งานได้บนหลายระบบปฏิบัติการ ทำให้สามารถ deploy ได้ตั้งแต่เครื่อง server ขนาดเล็ก ไปจนถึงระบบ cloud infrastructure

Windows

Orthanc มี binary package สำหรับ Windows รองรับทั้ง 32-bit และ 64-bit

Linux

รองรับ Linux distributions หลายตัว เช่น Ubuntu, Debian, Fedora และ Arch Linux

macOS

สามารถ build และ run Orthanc บน macOS สำหรับ development หรือ research

Docker

Orthanc มี official Docker images สำหรับ deploy ใน cloud หรือ container environment
