Technical Explanation

Introduction

"Orthanc เป็น lightweight standalone DICOM server พูดง่ายๆเหมือน database ที่ใช้สำหรับจัดเก็บและจัดการไฟล์ภาพทางการแพทย์ ใน demo นี้เราจะนำไฟล์ DICOM เข้าไปในระบบ จากนั้น Orthanc จะจัดเก็บไฟล์และแสดงข้อมูล metadata เช่น Patient, Study และ Series รวมถึงสามารถ preview ภาพทางการแพทย์ได้ผ่าน web interface"
