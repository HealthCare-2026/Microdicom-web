MicroDicom Viewer

Core Features

ฟีเจอร์สำคัญของ MicroDicom DICOM Viewer สำหรับการดู วิเคราะห์ และจัดการภาพทางการแพทย์

Reference: https://www.microdicom.com/

DICOM & DICOMDIR Import

Feature: เปิดและนำเข้าไฟล์ภาพทางการแพทย์ในรูปแบบ DICOM และไฟล์ DICOMDIR

Description: ระบบสามารถอ่านและโหลดชุดภาพทางการแพทย์จากไฟล์ DICOM หรือโครงสร้าง DICOMDIR ที่อยู่ในโฟลเดอร์หรือสื่อจัดเก็บข้อมูล เช่น CD, DVD หรือ USB

Example: ผู้ใช้สามารถเปิดภาพ CT Scan ของผู้ป่วยจากแผ่น CD ของโรงพยาบาลเพื่อดูข้อมูลการตรวจ

Image Viewing Tools

Feature: แสดงและปรับการมองเห็นภาพทางการแพทย์

Description: มีเครื่องมือสำหรับปรับภาพ เช่น Zoom, Pan, Window/Level, Brightness และ Contrast เพื่อช่วยให้มองเห็นรายละเอียดของโครงสร้างทางกายวิภาคได้ชัดเจนขึ้น

Example: แพทย์ปรับค่า Window/Level ในภาพ CT เพื่อแยกความแตกต่างระหว่างกระดูกและเนื้อเยื่อ

Measurement & Annotation

Feature: วัดและใส่คำอธิบายบนภาพทางการแพทย์

Description: รองรับการวัดระยะทาง มุม หรือพื้นที่ รวมถึงการเพิ่มข้อความ ลูกศร และสัญลักษณ์ต่าง ๆ บนภาพ

Example: แพทย์วัดขนาดของก้อนเนื้อในภาพ MRI และใส่เครื่องหมายเพื่อระบุตำแหน่งของรอยโรค

DICOM Anonymizer

Feature: ลบหรือซ่อนข้อมูลส่วนบุคคลของผู้ป่วย

Description: เครื่องมือนี้ช่วยปกป้องความเป็นส่วนตัวของผู้ป่วย โดยการลบข้อมูลที่สามารถระบุตัวตนได้ เช่น ชื่อผู้ป่วยหรือรหัสผู้ป่วย

Example: ก่อนนำภาพไปใช้ในการสอนหรือวิจัย สามารถลบข้อมูลผู้ป่วยออกจากไฟล์ DICOM

DICOM Tag Viewing & Editing

Feature: ดูและแก้ไขข้อมูล Metadata ของไฟล์ DICOM

Description: ผู้ใช้สามารถตรวจสอบข้อมูลสำคัญของภาพ เช่น Patient ID, Modality, Study Date และรายละเอียดการตรวจ

Example: นักวิจัยตรวจสอบข้อมูลวันที่ตรวจและประเภทเครื่องมือที่ใช้สร้างภาพ

Export

Feature: ส่งออกข้อมูลภาพทางการแพทย์ไปยังรูปแบบอื่น

Description: สามารถแปลงไฟล์ DICOM ไปเป็นรูปภาพ วิดีโอ หรือไฟล์ข้อมูล เช่น JPEG, PNG หรือ CSV เพื่อใช้ในการรายงานหรือการนำเสนอ

Example: แปลงภาพ CT Scan เป็นไฟล์ PNG เพื่อนำไปใส่ในรายงานทางการแพทย์

Multi-Planar Reconstruction

Feature: แสดงภาพในหลายระนาบของร่างกาย

Description: สามารถสร้างมุมมองภาพในระนาบต่าง ๆ เช่น Axial, Sagittal และ Coronal เพื่อช่วยให้เห็นโครงสร้างของอวัยวะได้ครบถ้วน

Example: แพทย์ดูภาพสมองจากหลายมุมเพื่อช่วยในการวินิจฉัยโรค

Image Fusion

Feature: รวมข้อมูลจากภาพทางการแพทย์หลายประเภท

Description: ช่วยรวมภาพจากการตรวจต่างชนิด เช่น PET และ CT เพื่อให้เห็นทั้งโครงสร้างของอวัยวะและข้อมูลการทำงานของเซลล์

Example: ใช้ตรวจหาตำแหน่งของเซลล์มะเร็งโดยดูจากภาพ PET-CT

Local Database

Feature: จัดเก็บและจัดการชุดภาพ DICOM ภายในเครื่อง

Description: สามารถนำเข้าภาพจากแหล่งต่าง ๆ เช่น CD, USB หรือโฟลเดอร์ในเครื่อง แล้วจัดเก็บไว้ในฐานข้อมูลภายในเพื่อค้นหาได้ง่าย

Example: โรงพยาบาลสามารถเก็บภาพตรวจของผู้ป่วยหลายรายไว้ในระบบเพื่อเปิดดูภายหลัง

PACS & DICOMweb Connectivity

Feature: เชื่อมต่อกับระบบจัดเก็บภาพทางการแพทย์ (PACS)

Description: สามารถค้นหา ดาวน์โหลด และส่งภาพทางการแพทย์ผ่านเครือข่ายด้วยโปรโตคอล DICOM และ DICOMweb

Example: แพทย์สามารถดึงภาพ MRI จากเซิร์ฟเวอร์ PACS ของโรงพยาบาลมาเปิดดูได้ทันที

Image Comparison

Feature: เปรียบเทียบภาพหลายภาพพร้อมกัน

Description: สามารถแสดงภาพหลายชุดในหน้าจอเดียวแบบแบ่งหน้าจอ เพื่อเปรียบเทียบภาพจากการตรวจต่างช่วงเวลา

Example: เปรียบเทียบภาพ X-ray ของผู้ป่วยก่อนและหลังการรักษา

Cine Loop Playback

Feature: แสดงภาพแบบต่อเนื่องเหมือนวิดีโอ

Description: ระบบสามารถเล่นภาพหลาย slice ต่อเนื่องกัน เพื่อดูการเคลื่อนไหวหรือการเปลี่ยนแปลงของภาพทางการแพทย์

Example: ใช้ดูภาพ ultrasound หรือเลื่อนดูภาพ CT scan ทีละชั้นอย่างต่อเนื่อง
